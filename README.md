# Red Hat Showroom Content Creation

Create professional Red Hat workshop and demo content using Claude Code skills.

## Quick Start

### Setup

1. Clone this repository:
```bash
git clone <repository-url>
cd <repository-name>
```

2. Start Claude Code session within the directory

3. List available skills:
```bash
/skills
```

### Create a Workshop Module

Run the `lab-module` skill (without `/`):
```
lab-module
```

Creates hands-on workshop modules with exercises, verification steps, and Red Hat style compliance.

### Create a Demo Module

Run the `demo-module` skill:
```
demo-module
```

Creates presenter-led demonstrations with business value messaging and Know/Show structure.

### Verify Content Quality

Run the `verify-content` skill:
```
verify-content
```

Checks content against Red Hat standards, technical accuracy, and accessibility requirements.
