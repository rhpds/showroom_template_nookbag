= Claude Code Agent Skills
:navtitle: Course Overview

This course teaches you to create, distribute, and compose Agent Skills for Claude Code. By the end you'll have built three skills, bundled them into a distributable marketplace, and used them in a multi-layer workflow alongside harness and workflow skills.

[IMPORTANT]
====
Before starting, ensure you have:

* Claude Code installed and working (`claude --version`)
* An Anthropic API key configured
* A GitHub account (for Module 4)
* Basic comfort with the terminal and git
====

== What You'll Build

Throughout this course, you'll create the **conference-toolkit** — a practical skill marketplace containing three complementary skills:

* **abstract-writer** — Drafts compelling conference talk abstracts based on your topic and target audience
* **conf-finder** — Discovers relevant conferences matching your interests and expertise
* **trip-report** — Generates structured post-conference writeups and key takeaways

These three skills will be bundled into a distributable marketplace called `conference-toolkit` that you can share and install across projects.

. xref:01-introduction.adoc[Introduction to Agent Skills] — What skills are, where they live, and how to use one from the Anthropic marketplace.

. xref:02-first-skill.adoc[Creating Your First Skill] — Hand-write `abstract-writer` from scratch, understanding skill anatomy and progressive disclosure.

. xref:03-advanced-skills.adoc[Advanced Skills with skill-creator] — Use Anthropic's `skill-creator` meta-skill to generate `conf-finder`, then refine it.

. xref:04-marketplaces.adoc[Marketplaces and Plugins] — Build `trip-report`, bundle all three skills into a `conference-toolkit` marketplace.

. xref:05-full-monty.adoc[The Full Monty] — Use superpowers + beads + your marketplace together in a real workflow.

. xref:06-bonus-programmatic.adoc[Bonus: Programmatic Skills] — (Coming soon) Using skills in Deep Agents and skills-calling-skills patterns.
